**TCG AI** adalah alat otomatisasi QA (Quality Assurance) berbasis web yang dirancang untuk mempercepat proses testing dan dokumentasi API. Project ini menggabungkan antarmuka modern dengan efisiensi backend untuk mendukung alur kerja developer dan tester.

## ✨ Fitur Utama

- **Modern UI/UX**: Menggunakan Tailwind CSS dengan dukungan Dark Mode yang nyaman di mata.
- **Glassmorphism Design**: Halaman About yang elegan dengan efek blur transparan.
- **Responsive Layout**: Optimal untuk tampilan desktop maupun mobile/tablet.
- **Integration Ready**: Dirancang untuk diintegrasikan dengan berbagai API contract dan automation script.

## 🛠️ Tech Stack

- **Frontend**: [Tailwind CSS](https://tailwindcss.com), FontAwesome Icons.
- **Backend**: PHP (Core).
- **Design Concept**: Glassmorphism & Modern Clean UI.

## 🚀 Instalasi & Penggunaan

1. **Clone repositori ini:**
   ```
   git clone https://github.com/muhamadsyafii/TCG-AI.git
2. **Pindahkan ke web server (XAMPP/Laragon/Nginx):** Pastikan folder project Anda berada di dalam direktori `htdocs` atau `www`.

3. **Akses melalui browser:** Buka browser dan arahkan ke alamat berikut:
http://localhost/[nama-folder]/index.php
